# Верстка интернет-магазина Drive Moto. Пет-проект.

Вёрстка 3-х страниц: 

    index.html
    catalog.html
    product.html     


Имеется: 

    адаптив.
    слайдеры.
    переключение по табам.
    модальные окна.
    кастомные чекбоксы, радиокнопки.
    ценовой range инпут.
    
Зависимости: 

    jQuery
    @fancyapps/fancybox
    ion-rangeslider
    jquery-validation
    rateyo
    slick-carousel
    

Запуск:

    1. npm i
    2. gulp
    3. gulp build - для билд сборки (опционально)
